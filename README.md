# 🎓 GPA Calculator Lab

> A complete web application for calculating Grade Point Average (GPA) with 3 progressive steps.

## 📊 GPA Interpretation

| GPA Range | Classification | Color |
|-----------|----------------|-------|
| ≥ 3.7 | Distinction 🎉 | 🟢 Green |
| 3.0 - 3.7 | Merit 👍 | 🔵 Blue |
| 2.0 - 3.0 | Pass ✅ | 🟡 Yellow |
| < 2.0 | Fail ❌ | 🔴 Red |
